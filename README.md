# RG Market Property Management System

A modern property management system for RG Market, a commercial building in Kigali, Rwanda.

## Tech Stack

- React 19
- TanStack Router
- TanStack Start
- Tailwind CSS
- TypeScript

## Development

```bash
npm install
npm run dev
```

## Deployment

This project is configured for deployment on Vercel.
